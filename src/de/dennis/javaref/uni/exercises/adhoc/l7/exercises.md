[ ] **Ad-hoc-Aufgabe 1**

Ändern Sie die Klasse SimpleFileServer von Abb. 14.2 so ab, dass keine Finally-
Klauseln mehr benötigt werden, aber trotzdem alle geöffneten Systemressourcen sicher
geschlossen werden, indem Sie konsequent die Try-with-Resources-Anweisung verwenden.
Das ist nicht ganz trivial und es kann sein, dass Sie dazu ein wenig experimentieren
müssen. Schauen Sie sich bei Problemen bitte auch die Fehlermeldungen Ihrer Entwicklungsumgebung
an. Sollten Sie auf größere Schwierigkeiten stoßen, wenden Sie sich gerne
per Moodle-Forum an die Kursbetreuung!
